## Package economics rule (Doc B)

Packages are **scoped retainers with hard caps** — not “sum of catalogue SKUs = unlimited monthly delivery.”

- À la carte (Top 8 + legacy menu) = **one-time / add-on** projects with explicit includes/excludes.
- Retainers = **monthly capacity** (posts, platforms, content units, support hours).
- Overage → quote as add-on or upgrade tier. Never imply unlimited delivery at Footprint/Growth prices.

## Affiliate economics (Doc B) — live public terms

| | |
|--|--|
| Recurring commission | **20–25%** by tier on active monthly retainers |
| Bronze (0–5 clients) | **20%** recurring |
| Silver (6–15) | **22%** + **$50** new-client bonus |
| Gold (16+) | **25%** + **$100** new-client bonus |
| Illustrative plans | Standard **$290**/mo → **$58** @20%; Professional **$790**/mo → **$197.50** @25% |
| Also commissionable | Footprint / Growth / Automation at same tier rates |
| Project SKUs | Commission on closed project fees at **Bronze floor (20%)** unless Gold partner negotiated |
| Payout | **15th** monthly · min **$100** · PayPal / bank / Stripe |

**Value-gap fix:** Affiliates sell **scoped packages**, not “$6,500 catalogue for $290.” Train: *what’s included this month* + *what upgrades*.

## Affiliate face (Doc A — Regional Hustler)

- **Who:** 25–45, urban SEA (Vientiane, Bangkok, …), mobile-first, relationship seller; often side income → full-time path.
- **Channels:** Facebook Messenger, LINE, WhatsApp first; short video + social proof; not desktop-heavy forms.
- **Wants:** Clear % commissions, reliable 15th pay, ready creatives, simple talking points, peer community.
- **Avoid:** Jargon, multi-step onboarding, long payment cycles, complex bonus math.
- **Laos pitch:** Tangible ROI (more bookings, fewer FB DMs, professional presence).  
- **Thai pitch:** Growth/automation + LINE/TikTok awareness.

**Promise:** *You own the relationship — we deliver.*
