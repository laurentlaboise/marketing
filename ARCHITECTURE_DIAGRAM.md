## Key Integration Points

1. **Form Submission**
   - Hidden field `article_images` contains JSON string
   - Server parses and validates before saving
   - Database stores as JSONB

2. **Image Library API**
   - Authenticated endpoint
   - Paginated responses
   - Search and filter support

3. **Modal Communication**
   - Vanilla JavaScript (no framework)
   - Event-driven architecture
   - State management in memory

4. **Data Persistence**
   - JSONB column in PostgreSQL
   - Automatic JSON serialization
   - Backward compatible default value
